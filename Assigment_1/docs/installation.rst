Installation
============

Clone the repository:

.. code-block:: bash

   git clone https://github.com/juansta02/Artificial-Intelligence-And-Open-Science-In-Research-Software-Engineering.git
   cd Artificial-Intelligence-And-Open-Science-In-Research-Software-Engineering

Create environment:

.. code-block:: bash

   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
